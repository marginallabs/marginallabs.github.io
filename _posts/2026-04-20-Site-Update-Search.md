# What Changed

The blog listing page now supports full-text search with boolean operators, category filter buttons, and sort options (newest, oldest, shortest read, longest read).

# How to Use

Type keywords into the search bar. Use quotes for exact phrases and a minus sign to exclude terms. Click category buttons to filter by topic.
