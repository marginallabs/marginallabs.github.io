# The Endogeneity Problem

Regression assumes the explanatory variable is uncorrelated with the error. When this fails, OLS is biased and inconsistent.

# The IV Solution

A valid instrument is correlated with the endogenous regressor but uncorrelated with the error term. This exclusion restriction is the key assumption.
